- 명확히 구분된 순수 함수

- 모든 입력값을 매개변수로써 받아들여야 한다. → 예측 가능한 함수
- 재사용 가능해야 한다.
- 👍
  - 코드가 읽기 쉽고 잘 정돈 되어 있다.
